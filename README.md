# Projeto-web
 🏍️ *Speed Motors*

 Documentação do Projeto

O Speed Motors é um site simples de motos, utilizando HTML, CSS e JavaScript.

A ideia do projeto é criar um site organizado e fácil de usar, mostrando motos para venda com tela de login e páginas funcionando.


*Funcionalidades*

* Tela de login
* Página de motos
* Botões funcionando
* Navegação entre páginas
* Validação dos campos
* Mensagens de alerta

*Requisitos Funcionais (RF)*

* O sistema deve permitir login do usuário.
* O sistema deve mostrar motos na tela.
* O sistema deve permitir mudar de página.
* O sistema deve validar campos vazios.
* O sistema deve possuir botões funcionando.


*Requisitos Não Funcionais (RNF)*

* O site deve ser simples de usar.
* O sistema deve carregar rápido.
* O código deve ficar organizado.

 *Regras de Negócio (RN)*

* O usuário deve preencher email e senha.
* O sistema não deve aceitar campos vazios.
* Apenas usuários logados podem entrar.
* As motos devem mostrar nome e preço.
